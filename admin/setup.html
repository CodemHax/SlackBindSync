ü§ñ Telegram Bot Setup

1 Create a Bot with BotFather
  • Open Telegram and search for @BotFather
  • Send the command /newbot
  • Follow the prompts to choose a name and username for your bot
  • BotFather will give you a Bot Token - copy this token
Token Format: The token will look like 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
Structure: [BOT_ID]:[AUTH_TOKEN]
‚ö†Ô∏è Keep this token secure! Anyone with this token can control your bot.
2 Create a Channel or Group
  • Create a new Telegram channel or group
  • Add your bot as an administrator
  • Make sure the bot has permission to read and send messages
3 Get the Chat ID

Method 1: Using a Bot

  • Add @userinfobot to your channel/group
  • It will automatically display the Chat ID
  • Copy the Chat ID (it will be a negative number for groups/channels)

Method 2: Using API

  • Send a message in your channel/group
  • Visit: https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
  • Replace YOUR_BOT_TOKEN with your actual token
  • Look for "chat":{"id":-1001234567890} in the response
Example Chat ID: -100XXXXXXXXXX (for groups/channels)
4 Add to .env File

Add these lines to your .env file:

TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
TELEGRAM_CHAT_ID=-1001234567890
‚úÖ Done! Your Telegram bot is now configured. Messages will sync between Telegram and other platforms.

üéÆ Discord Bot Setup

1 Create a Discord Application
  • Go to Discord Developer Portal
  • Click "New Application"
  • Enter a name for your application (e.g., "BindSync Bot")
  • Click "Create"
2 Create a Bot User
  • In your application, go to the "Bot" section in the left sidebar
  • Click "Add Bot" and confirm
  • Under the bot's username, click "Reset Token" and copy it
  • This is your DISCORD_BOT_TOKEN
Token Format: YOUR_BOT_TOKEN_HERE
Structure: [USER_ID_BASE64].[TIMESTAMP].[HMAC]
‚ö†Ô∏è Keep this token secret! Never share it or commit it to version control. If exposed, regenerate it immediately.
3 Enable Required Intents

In the Bot section, scroll down to "Privileged Gateway Intents":

  • Enable "Message Content Intent"
  • This allows your bot to read message content
4 Invite Bot to Your Server
  • Go to the "OAuth2" ‚Üí "URL Generator" section
  • Under "Scopes", select:
    • bot
  • Under "Bot Permissions", select:
    • Read Messages/View Channels
    • Send Messages
    • Read Message History
  • Copy the generated URL and open it in your browser
  • Select your server and authorize the bot
5 Get Channel ID
  • In Discord, go to Settings ‚Üí Advanced
  • Enable "Developer Mode"
  • Right-click on the channel you want to use
  • Click "Copy ID"
  • This is your DISCORD_CHANNEL_ID
Example Channel ID: XXXXXXXXXXXXXXXXXX
6 Add to .env File

Add these lines to your .env file:

DISCORD_BOT_TOKEN=YOUR_BOT_TOKEN_HERE
DISCORD_CHANNEL_ID=XXXXXXXXXXXXXXXXXX
‚úÖ Done! Your Discord bot is now configured. Messages will sync between Discord and other platforms.

üíº Slack Bot Setup

1 Create a Slack App
  • Go to Slack API
  • Click "Create New App"
  • Choose "From scratch"
  • Enter an app name (e.g., "BindSync Bot")
  • Select your workspace
  • Click "Create App"
2 Enable Socket Mode
  • In your app settings, go to "Socket Mode"
  • Toggle "Enable Socket Mode" to ON
  • Generate an app-level token:
    • Enter a token name (e.g., "BindSync Socket Token")
    • Add the connections:write scope
    • Click "Generate"
  • Copy the App Token (starts with xapp-)
  • This is your SLACK_APP_TOKEN
App Token Format: xapp-1-XXXXXXXXXXX-XXXXXXXXXXXX-XXXXXXXX...
Starts with: xapp- (App-level token)
3 Configure Bot Token Scopes
  • Go to "OAuth & Permissions"
  • Scroll to "Bot Token Scopes"
  • Add these scopes:
    • chat:write - Send messages
    • channels:history - Read channel messages
    • channels:read - View channel info
    • users:read - Get user information
4 Install App to Workspace
  • Scroll to the top of "OAuth & Permissions"
  • Click "Install to Workspace"
  • Review permissions and click "Allow"
  • Copy the Bot User OAuth Token (starts with xoxb-)
  • This is your SLACK_BOT_TOKEN
Bot Token Format: xoxb-XXXXXXXXXXXX-XXXXXXXXXXXX-XXXXXXXX...
Starts with: xoxb- (Bot user token)
‚ö†Ô∏è Security Note: Treat both tokens as passwords. Regenerate immediately if compromised.
5 Enable Event Subscriptions
  • Go to "Event Subscriptions"
  • Toggle "Enable Events" to ON
  • Under "Subscribe to bot events", add:
    • message.channels - Listen to messages in channels
  • Click "Save Changes"
6 Get Channel ID
  • Open Slack and navigate to your channel
  • Click the channel name at the top
  • Scroll down in the About tab
  • At the bottom, you'll see the Channel ID
  • Copy this ID (e.g., C01234ABCDE)
Alternative: Right-click the channel ‚Üí Copy link. The ID is at the end of the URL.
7 Invite Bot to Channel
  • In your Slack channel, type: /invite @YourBotName
  • Or click channel name ‚Üí Integrations ‚Üí Add apps
8 Add to .env File

Add these lines to your .env file:

SLACK_BOT_TOKEN=xoxb-XXXXXXXXXXXX-XXXXXXXXXXXX-XXXXXXXXXXXXXXXXXXXXXXXX
SLACK_APP_TOKEN=xapp-1-XXXXXXXXXXX-XXXXXXXXXXXX-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
SLACK_CHANNEL_ID=C0XXXXXXXXX
‚úÖ Done! Your Slack bot is now configured. Messages will sync between Slack and other platforms.
